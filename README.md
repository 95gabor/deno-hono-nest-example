# Platform Hono sample

## Tools

- [nestjs](https://nestjs.com/)
- [Hono](https://hono.dev/)
- [@uki00a/nestjs-platform-hono (nestjs hono adapter)](https://github.com/uki00a/nestjs-platform-deno)

## Server usage

```bash
npm install
npm run build
npm start
```

## Serverless usage

```bash
npm install
npm run build:lambda
npm run test:lambda
```
