# Platform Hono sample

## Tools

- [nestjs](https://nestjs.com/)
- [Hono](https://hono.dev/)
- [@kiyasov/platform-hono (nestjs hono adapter)](https://github.com/kiyasov/platform-hono)
- [esbuild](https://esbuild.github.io/)

## Server usage

```bash
npm install
npm run build
npm start
```

## Serverless usage

```bash
npm install
npm run build:lambda
npm run test:lambda
```
